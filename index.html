Hello there,

Welcome! This is Trinadh, I code to pay my bills. I started my career as a Java Developer alongside building web pages using HTML. It's the time when the industry started incorporating UI frameworks for building web applications so I started using jQuery. Very soon there is this another framework called Angular and I got a chance to work on it for a good amount of time. I recently started using React as well. I can work on both front-end as well as back-end technologies.

In a nut shell, I use Java and its frameworks such as Spring Boot to create web services and get the data from servers, and use Angular or React to tell browsers how to show that data in a meaningful way. Hey, did I tell you that I work on Node.js? If not.. Yes, I do know how to build applications using Node.js. Confused what's my role is? You can call me Full Stack Java | Web Developer.
